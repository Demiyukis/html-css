Para poder controlar onde o site de destino vai abrir, podemos usar os atributos target que suporta os seguintes valores:

  • _blank: Abre o link em uma nova janela ou aba
  • _self: Abre o link na mesma janela ou aba
  • _parent: Abre o link no frame pai
  • _top: Abre o link no documento pai
  • nome: Abre o link no frame com o nome especificado

Para indicar qual é a natureza do destino usando o atributo rel que aceitam vários valores, entre eles:

  • next: Indica que o link é para a próxima parte do documento atual
  • prev: Indica que o link é para a parte anterior do documento atual
  • author: Indica que é um link para o site do autor do artigo atual
  • external: Indica que é um link para outro site que não faz parte do site atual
  • nofollow: Indica que é um link para o site não endossado, como um link pago

Links de download

Existem links para efetuar downloads de materiais como PDFs, arquivos ZIP. Basta fazer o link diretamente para o arquivo que se deseja fazer download e adicionar o atributo download com o valor configurado para o nome do arquivo a ser baixado e o atributo type para indicar ao navegador que tipo de arquivo está sendo baixado.

Beixe o arquivo PDF do meu livro

Seguem alguns media types usados no nosso dia-a-dia:

  • text/plain: Arquivos de texto simples
  • text/html: Arquivos HTML
  • text/css: Arquivos CSS
  • text/javascript: Arquivos JavaScript
  • image/jpeg: Imagens JPEG
  • image/png: Imagens PNG
  • image/gif: Imagens GIF
  • audio/mpeg: Arquivos de áudio MP3
  • video/mp4: Arquivos de vídeo MP4
  • application/pdf: Arquivos PDF
  • application/zip: Arquivos ZIP
